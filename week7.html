Project information

  • Sprint-7: Learning Objectives page & Day Modules page
  • User Story:As a course developer, I should be able to link the pre-requisite learning objectives for a learning objective, so that it can be used to personalize the module for each student in the iLMS.
  • Description:

    • In this week, we worked on learning objective page and day module page. In learning objective page, when the user clicks on the dropdown button then it displays the items like preview and delete. When user clicks on preview then it displays the id, title, activity type, video id, notes title and when the user clicks on delete then that file gets deleted. When we land in learning objective page then in the right of the page the files created will be displayed. Here we have used api calls to get data from backend.

    In daywise modules, we have created a form which consists of id and description and given a two select dropdowns where one is for learning objective and other is for prerequisites. In the select dropdown, the learning objectives were fetched from backend using api calls. After selecting the learning objectives then we need to click on add and to preview that we have displayed it in table below. After entering all the details we need to click on save button. After clicking on it then the json get created and that json get appended to the file. That files will be stored in one folder which is created based on name of the course. It is previewed at the right of the page and clicking on it, it displays the items like preview and delete. When we click on preview then the details in the file gets displayed. When we click on delete then that files get deleted.

    Created the buttons like course page and day module page in learning objectives page. When user clicks on course page then the page gets redirects to course page and when user clicks on module page then it redirects to day modules page. In day modules page we created a button and when user clicks on it then the page redirects to learning objectives page. In this way, we integrated all the screens.
